# Django Portfolio Website

This is my personal portfolio website built with Django.

## Features

* Home Page
* About Page
* Contact Page
* Responsive Design

## Tech Stack

* Python
* Django
* HTML5
* CSS3
* Bootstrap

## Installation

```bash
git clone https://github.com/thakurshubhamchauhan/django-portfolio.git
cd django-portfolio
pip install django
python manage.py runserver
```
